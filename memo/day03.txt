0. 복습
리스트 태그
<ol></ol> : 순서 있는 리스트를 만든다
<ul></ul> : 순서 없는 리스트를 만든다
<li></li> : ol 또는 ul 태그 내부에서 사용

테이블 태그(표 만들기)
<table></table> : 표를 만드는 테그
<caption></caption> : 해당 표의 제목
<tr></tr> : 행을 만드는 태그
<td></td> : 열을 만드는 태그
	colspan, rowspan
<th></th> : 제목행에 컬럼을 만드는 태그 

이미지 삽입 태그
<img src = "파일경로" alt = "설명" >
src 속성은 이미지 경로, 웹 상의 주소 
상대경로 : 현재 내 위치 기반으로 이동하는 경로
./ : 현재 폴더 ../ : 상위 폴더

링크를 거는 a태그(앵커 태그)
<a href="링크 주소나 경로"></a>
a태그로 감싼 요소는 해당 경로로 링크가 걸린다 

인라인/블록 요소

<span></span> : 대표적인 인라인 태그
<div></div> : 대표적인 블록 태그 
- span과 div태그는 사이트 화면의 레이아웃을 나누는데 주로 사용되며 매우 중요하다 
- 둘다 의미가 없다. 


인라인 요소
	- 내용물의 크기만큼만 영역을 차지하므로 다른 요소와 같은 줄에 위치할 수 있다
	즉, 줄 바꿈이 되지 않는다
	- span, img, strong 등이 있다

블록 요소
	- 내용물과 상관없이 한 줄을 다 차지하므로 다른 요소와 같은 줄에 위치할 수 없다. 
	즉, 너비 속성이 100% 다 차지하고 자동으로 줄 바꿈이 된다.
	- 다른 요소를 하나의 블록으로 묶어주기 위해 사용한다.
	- 블록요소는 직사각형 모양이다
	-h1, div, p 등이 있다.

CSS(Cascading Style Sheets)
HTML로 만든 웹 문서의 스타일(디자인)을 담당한다 
스타일링이란 글꼴, 색상, 테두리, 정렬, 배치(레이아웃) 방법등의 외관을 말한다 
HTML에서 제공하는 속성으로도 간단한 스타일링을 지정할 수 있다.
그러나 내용과 디자인이 분리되어 있어야 유지보수에 유리하므로 CSS를 사용하는 것이 더 좋다. 

CSS 형식
속성1 : 속성값;
속성2 : 속성값; 
속성은 미리 정해져 있으며 필요에 따라 검색해서 사용한다.

스타일 적용하는 방법 
스타일을 적용시키는 방법은 3가지가 있다.

1. 인라인 스타일
	스타일을 적용시킬 때 태그에 style 속성을 사용한다
	<p style="color : red; background-color :black">내용</p>

	인라인 스타일은 간단하게 사용하기에는 좋으나 코드가
	길어질 수록 가독성이 떨어지며, css와 html이 뒤섞여 유지보수가 매우 힘들어진다.

2. 내부 스타일
	<head> 영역에 <style></style> 태그를 사용하여 각 요소들의 스타일을 정의한다.
	
	형식은 다음과 같다.
	선택자 {
		속성1 : 속성값;
		속성2 : 속성값;
	}
	선택자 : 어떤 요소를 선택하기 위해 사용하는 것 

id와 class
id와 class 속성은 특정들을 식별하고 그룹화하는데 사용한다.
모든 테그에 사용 가능한 속성이다

id
특정 태그에 식별 가능한 고유한 값을 부여함
선택자로 사용할 때는 #id로 사용한다 

class
여러 태그를 하나의 집단으로 묶어줌
선택자로 사용할때는 .class로 사용한다

하나의 태그에 id와 class를 동시에 가질 수 있으며 class를 여러개 가질 수도 있다
주로 class를 많이 사용하며 id는 그 페이지에서 중요하고 유일한 요소에게 부여하는 것이 좋다

3. 외부 스타일
	별도의 css 파일을 만들어 적용시킨다
	css와 html을 완전히 분리하여 관리할 수 있기 때문에 가장 많이 사용하는 방식이다
	css파일 html에 적용시키기 위해서는 head 영역에 link태그를 사용한다
	<link rel="stylesheet" href="파일 경로">
	rel 속성은 현재 페이지와의 관계를 의미하며, href는 적용시킬 css파일의 
	경로를 적어준다.
