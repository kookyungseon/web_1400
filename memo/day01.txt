HTML
: Hypertext Markup Language : 웹 문서를 만드는 언어
   웹 사이트 화면을 구성하는 것들 중 뼈대의 역할을 한다.
   태그를 사용하여 웹 페이지가 어떤 구조로 만들어져 있는지 알려준다
   태그란 해당 페이지에 관련된 정보를 나타낸 것으로 <>를 사용한다.

HTML, CSS
   웹 개발에서 가각 문서구조와 스타일링을 담당하는 두 가지 핵심 언어

HTM
   웹페이지의 구조와 콘텐츠를 정의하는 마크업 언어(뼈대)

CSS(Cascading Style Sheets)
   HTMl 문서의 시작적인 표현을 담당하는 스타일 시트 언어(디자인, 스타일)

HTML 구조
   <html>
      <head>
      이 페이지에 관련된 메타 정보
      </head>
      <body>
      본문 -> 실제 웹 페이지에 나타나는 내용
      </body>
   </html>

<title> : 이 페이지의 타이틀(head 태그 안에서 사용) -> 북마크로 사용됨
<h1>, <h2>,...<h6> : 제목을 의미(숫자가 작을수록 큰 글씨로 나타난다)
<p> : 단락을 의미(줄바꿈이 됨)


HTML
   웹페이지를 만드는 문법을 갖춘 언어

   태그(tag)로 이루어져 있음
   태그 : 어떤 의미를 지니고 있다
   여는태그(Opening Tag) : 요소의 이름과 열고닫는 꺽쇠 괄호로 구성
   요소의 시작부터 효과가 적용되기 시작함
   닫는태그(Closing Tag) : 요소의 이름 앞에 슬래쉬(/)를 써서 구성
   요소의 끝에 위치함
   내용(Content) : 요소의 내용
   요소(Element) : 여는 태그, 닫는 태그, 내용을 통틀 어 요소라고 함

5. 태그의 종류
	<h1></h1> ~ <h6></h6> : 제목 태그, 숫자가 작을수록 굵어지고 커진다
	<p></p> : 단락 태그, 단락 위/아래 간격이 벌어지며 자동으로 줄이 바뀐다
	<strong></strong> , <b></b> : 굵은 글씨
	<br> : 줄 바꿈 태그, 닫는 태그가 없다
	<hr> : 가로선을 그려주는 태그, 닫는 태그가 없다
	닫는 태그가 없는 것을 빈태그라고 부른다
